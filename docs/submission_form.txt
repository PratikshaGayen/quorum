FIELD 2 — USE CASE SUMMARY
--------------------------
Quorum is a verifiable decision engine for high-stakes KYC / onboarding
approvals. Instead of one LLM making an approve/reject call — which
hallucinates, is overconfident, and leaves no record of why — Quorum runs a
jury of three diverse LLM personas (Compliance Officer, Risk Analyst, Devil's
Advocate) that deliberate each case in parallel on a single AMD Instinct GPU.
Each juror returns a structured verdict (decision + confidence + rationale +
risk flags); a deterministic adjudicator combines them into a final
APPROVE / REJECT / ESCALATE with a panel-confidence score and a verbatim
dissent report — so every decision ships with an audit trail.

Results (real run on 1 x AMD Instinct MI300X, Qwen2.5-7B-Instruct on vLLM/ROCm,
151 synthesized labeled KYC cases): the council lifts recall on bad actors from
0.43 to 0.96 while precision stays 1.0 — catching 43 cases the confident single
model got wrong (contradiction-catch rate 0.285), and escalating only the
genuinely ambiguous ~19% to a human. The entire jury runs concurrently on one
GPU at ~1.55x the throughput of running the models one at a time (~86% of
192 GiB VRAM). Built fully during the hackathon; open-weight models; the
dataset is synthetic and non-proprietary (synthesis method in the repo).
